# Информационная система тестирования знаний сотрудников

Простой учебный веб-проект на Flask для темы диплома:
«Разработка информационной системы для тестирования знаний сотрудников организации».

## что есть в системе

- вход пользователей по логину и паролю;
- роли: администратор, специалист отдела кадров, сотрудник;
- создание тестов;
- добавление вопросов и вариантов ответов;
- назначение тестов сотрудникам;
- прохождение теста сотрудником;
- автоматический расчет результата;
- отчет по прохождению тестов.

## Тестовые пользователи

| --- | --- | --- |
| Администратор | `admin` | `admin123` |

| Сотрудник | `petrova` | `12345` |

## Как запустить локально

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

После запуска открыть:

```text
http://127.0.0.1:5000
```

База SQLite создается автоматически при первом запуске.

## Деплой на Render

1. Загрузить проект в GitHub.
2. На Render создать новый `Web Service`.
3. Подключить репозиторий.
4. Выбрать Python.
5. Build Command:

```bash
pip install -r requirements.txt
```

6. Start Command:

```bash
gunicorn app:app
```

Файл `render.yaml` уже добавлен, поэтому Render может взять настройки автоматически.
